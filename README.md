# 💸 Pocket Pilot – Expense Tracker App

Pocket Pilot is a Flutter-based expense tracking app built to manage daily finances in a simple and intuitive way. It focuses on clean UI, useful insights and a smooth user experience.

---

## 🚀 Features

- Add, edit and delete transactions  
- Track both income and expenses  
- Categorize spending (Food, Transport, Shopping, etc.)  
- View all transactions with filters  
- Clean and modern UI  

---

## 💱 Multi-Currency Support

- Supports INR, USD and EUR  
- Stores all values internally in INR  
- Converts values dynamically based on selected currency  
- Keeps calculations accurate across the app  

---

## 📊 Insights

- Pie chart for spending visualization  
- Category-wise expense breakdown  
- Total income and expense overview  

---

## 🎯 Filters

- Filter by transaction type (All / Income / Expense)  
- Filter by category  
- Helps analyze specific transactions easily  

---

## ⚙️ Settings

- Change username  
- Toggle dark mode  
- Select preferred currency  
- Reset all transaction data  

---

## 🎨 UI

- Dark mode support  
- Consistent design across all screens  
- Smooth and minimal user experience  

---

## 🛠 Tech Stack

- Flutter  
- Hive (local database)  
- SharedPreferences  
- fl_chart  
- Google Fonts  

---

## 🧠 How it works

- All transaction data is stored in INR  
- Values are converted only when displayed  

Example:  
₹830 → $10 (USD)
